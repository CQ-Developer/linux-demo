# 命令: 显示用户拥有的用户组
groups

# 命令: 修改用户的有效组
newgrp

# 命令: 创建用户
useradd

# 命令: 修改密码
passwd

# 命令: 修改密码过期信息
chage

# 命令: 修改账户信息
usermod

# 命令: 删除账户
userdel

# 命令: 查看用户GID和UID
id

# 命令: 修改用户详细信息
chfn

# 命令: 修改用户的默认shell
chsh

# 命令: 创建用户组
groupadd

# 命令: 修改组属性
groupmod

# 命令: 删除组
groupdel

# 命令: 组管理员
gpasswd

# ACL: Access Control List
访问控制列表的作用是对某个用户进行特别的权限设置。
对于单机使用基本用不到。
作为了解只需要看两个命令的就好: getfacl, setfacl

su

# 命令: 以其他用户身份执行命令
