top - 动态的查询系统中存在的进程和它们的资源消耗情况

# 用法
# 空格和-都是可以省略的
# top -hv | -bcEHiOSs1 -d secs -n max -u|U user -p pid(s) -o field -w [cols]
-------------------------------------------------------------------------------
-h  :显示帮助信息
-v  :显示版本信息
-------------------------------------------------------------------------------
-b         :进入批量模式, 适合于将top的结果传递给其他程序或文件，通常和-n配合
-n number  :指定执行几次结束top
-------------------------------------------------------------------------------
-d secs  :几秒刷新一次
-------------------------------------------------------------------------------
-c  :展示触发进程的命令行或程序名，可以通过交互命令c修改
-------------------------------------------------------------------------------
-e k|m|g|t|p  :使用指定单位展示任务区的内存，可以通过交互命令e修改
-E k|m|g|t|p  :使用指定单位展示摘要区的内存，可以通过交互命令E修改
-------------------------------------------------------------------------------
-H  :进入线程模式, 可以通过交互式命令H修改
-------------------------------------------------------------------------------
-o fieldname  :根据指定的fieldname对任务进行排序, +号降序, -号升序
-O            :显示有哪些字段可用
-------------------------------------------------------------------------------
-p<PID,PID...>  :监控指定的PID, 该选项和-u和-U互斥
-------------------------------------------------------------------------------
-u user/UID  :匹配有效用户, 用!号取反
-U user/UID  :匹配任何用户, 用!号取反
