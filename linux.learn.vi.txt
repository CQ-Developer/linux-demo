# 光标
上 [k]
下 [j]
左 [h]
右 [l]

# 翻页
上一页 [Ctrl+b]
下一页 [Ctrl+f]
上半页 [Ctrl+u]
下半页 [Ctrl+d]

# 光标跳跃移动
跳到非空格的下一行 [+]
跳到非空格的上一行 [-]

# 光标行内移动
向右移动n个字符    [n<Space>]
移动到第一个字符   [0]
移动到最后一个字符 [$]

# 屏幕内移动
屏幕内第一行第一个字符 [H]
屏幕内中间行第一个字符 [M]
屏幕内最底行第一个字符 [L]

# 光标全局移动
光标移动到第一行   [gg]
光标移动到最后一行 [G]
光标移动到第n行    [nG]
向下移动n行        [n<Enter>]

# 查找
在光标的下方向下查找word [/word]
在光标的上方向上查找word [?word]
查找下一个 [n]
查找上一个 [N]

# 替换
在第n1行到第n2行之间查找word1并替换为word2            [:n1,n2s/word1/word/g]
在第n1行到第n2行之间查找word1并替换为word2并需要确认  [:n1,n2s/word1/word/gc]
在第1行到最后一行之间查找word1并替换为word2           [:1,$s/word1/word2/g]

# 删除
删除光标所在处的前一个字符     [x]
删除光标所在处的后一个字符     [X]
删除光标后的n个字符            [nx]
删除光标所在行                 [dd]
删除光标所在及向下共n行        [ndd]
删除光标所在处到行首的所有字符 [d0]
删除光标所在处到行末的所有字符 [d$]
删除光标所在行到第一行         [d1G]
删除光标所在行到最后一行       [dG]

# 复制
复制光标所在行                 [yy]
复制光标所在及向下共n行        [nyy]
复制光标所在处到行首的所有字符 [y0]
复制光标所在处到行末的所有字符 [y$]
复制光标所在行到第一行         [y1G]
复制光标所在行到最后一行       [yG]

# 粘贴
复制到光标所在的下一行 [p]
复制到光标所在的上一行 [P]

# 其他
将光标所在行及下一行合并 [J]

# 上一个操作
恢复上一个操作 [u]
重做上一个操作 [Ctrl+r]
重复上一个操作 [.]

# 进入编辑模式
在光标所在处输入         [i]
在光标所在行的行首处输入 [I]

# 进入编辑模式
在光标所在的下一个字符输入 [a]
在光标所在行的行尾处输入   [A]

# 进入编辑模式
在光标所在行之下新起一行进入输入 [o]
在光标所在行之上新起一行进入输入 [O]

# 进入编辑模式
替换光标所在处的一个字符     [r]
持续替换光标所在处的一个字符 [R]

# 保存与退出
保存     [:w]
强制保存 [:w!]
退出     [:q]
强制退出 [:q!]
保存退出 [:wq] [ZZ]

# 写出和读入新文件
将当前文件写出到新文件 [:w filename]
将n1到n2行的内容写出到新文件 [:n1,n2 w filename]
在光标所在行的下一行中插入其他文件的内容 [:r filename]

# 命令
暂时退出vi执行linux命令 [:! command]

# 反白选择
将光标经过的地方反白选中 [v]
将光标经过的行反白选中   [V]
将光标经过的字符反白选中 [Ctrl+v]
复制反白区域 [y]
粘贴复制区域 [p]
删除反白区域 [d]

# 多文件编辑
同时编辑多个文件   [vim file1 file2 file3 ...]
显示开启的所有文件 [:files]
编辑下一个文件 [:n]
编辑上一个文件 [:N]

# 多窗口
新建一个窗口打开当前文件 [:sp]
新建一个窗口打开指定文件 [:sp filename]
移动到下方窗口 [Ctrl+w+j]
移动到上方窗口 [Ctrl+w+k]
关闭光标所在窗口 [:q]

# 关键词补全
以文件扩展名和内置的关键词进行补全         [Ctrl+x+o]
以当前目录内的文件名作文关键字进行补全     [Ctrl+x+f]
以正在编辑文件的内容文字作为关键词进行补全 [Ctrl+x+n]

# 日志文件
记录曾经做过的操作 ~/.viminfo
vim操作环境自定义配置文件 ~/.vimrc

# 环境设置参数
显示非默认环境参数设置值 [:set]
显示所有环境参数设置值   [:set all]
显示行号 [:set nu]
隐藏行号 [:set nonu]
搜索结果开启高亮显示 [:set hlsearch]
搜索结果关闭高亮显示 [:set nohlsearch]
开启自动缩进 [:set autoindent]
关闭自动缩进 [:set noautoindent]
开启自动保存备份文件 [:set backup]
关闭自动保存备份文件 [:set nobackup]
显示右下角状态栏 [:set ruler]
关闭右下角状态栏 [:set noruler]
显示左下角状态栏 [:set showmode]
关闭左下角状态栏 [:set noshowmode]
开启程序相关语法颜色显示 [:syntax on]
关闭程序相关语法颜色显示 [:syntax off]
设置深色背景色调 [:set bg=dark]
设置浅色背景色调 [:set bg=light]
设置退格键功能 [:set backspace=0|1|2]
